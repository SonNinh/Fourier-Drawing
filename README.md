# Introduction
This repository demonstrates application of Fourier Serie on drawing any 2D line-art painting
# Requirement
```
opencv-python                 4.5.5.62
numpy                         1.22.3
svg.path                      6.2
```
# How to run

```bash
python main.py --svg svg/cat_2.svg --ppp 0.5
```
